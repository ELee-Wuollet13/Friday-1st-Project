The week in review:

  • The rapid changes in learning pace have been a bit jarring
  • On my first day I felt like I was picking git up slowly, but HTML made sense
  • On the second day I worked with two more experiences partners and we practiced Git pushing while learning new HTML machanics and to a certain extent the pace and my own newness to coding kept me from retaining some of the new elements
  • Day three I worked with another pair of more advanced students and while I enjoyed getting to know them I will be revisiting some of the elements we worked on
  • Overall I feel my grasp of Git and basic HTML are strong, with more complex HTML and CSS requiring some review
